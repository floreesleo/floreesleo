### 💼 Experiencias:
#### Nombre del Proyecto:
Aplicación web de gestión documental y publicación de articulos para el colegio de arquitectos chiapanecos sección comitán.

#### Descripción del Proyecto:
Desarrollé una aplicación web integral para un colegio de arquitectos en Comitán de Dominguez. La aplicación permite a los usuarios acceder a publicaciones del colegio y cuenta con un sistema de gestión de documentos privados para cada miembro.

#### Tecnologías Utilizadas:
- Desarrollo Frontend: React JS.
- Base de Datos: Supabase.
- Herramientas: Git, Figma.

#### Roles y Responsabilidades:
- Diseñé y desarrollé la interfaz de usuario (UI) para mostrar las publicaciones del colegio.
- Implementé un sistema de gestión de documentos privados utilizando storage de Supabase.
- Diseñé en el desarrollo del backend para el registro de usuarios, creación, edición y eliminación de publicaciones.

#### Resultados Destacados:
- Éxito en la implementación de un sistema eficiente de gestión de documentos privados para los usuarios.
- Contribución al aumento de la accesibilidad a la información del colegio mediante la presentación de publicaciones de manera organizada.

#### Lecciones Aprendidas:
- Aprendí a trabajar en un proyecto de desarrollo web completo, desde el diseño hasta la implementación.
- Mejoré mis habilidades de colaboración y comunicación al trabajar en el desarrollo del proyecto.
